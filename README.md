## Features

fast format your selected snippets, to create your code snippets easily in vscode

formatted snippets eg:

```
"your description": {
		"scope": "",
		"prefix": "your prefix",
		"body": [
			"50%{",
				"left:calc(100% - 95px);",
				"width:60px;",
				"height:60px;",
			"}"
		],
		"description": "Log output to console"
	}
```
